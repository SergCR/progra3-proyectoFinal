Tabla usuario

Metodo de usuario URL Atributos Descripcion
GET localhost:8081/getListaUsuarios >ListaUsuarios Imprime la lista de usuarios
GET localhost:8081/getUsuario String email Trae un usuario en específico usando el email
POST localhost:8081/addUsuario String nombre, String apellido, String email, String password Añade un usuario
UPDATE localhost:8081/updateUsuario Integer userid, String nombre, String apellido, String email, String password Actualiza la informacion del usuario
DELETE localhost:8081/deleteUsuario String email Elimina el usuario

Tabla notas

Metodo de Notas URL Atributos Descripcion
GET localhost:8081/addNotas int userID, String textoNota, String tituloNota Agrega notas
GET localhost:8081/getAllNotas AllNotas Trae todas las notas
GET localhost:8081/getNotasDeUsuarios String userEmail Trae una nota en específicos
UPDATE localhost:8081/updateNota Integer noteID, String texto, String titulo Actualiza la informacion de la nota
DELETE localhost:8081/deleteNota Integer noteID Elimina la nota
